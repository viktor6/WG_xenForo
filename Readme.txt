Цель создания данного файла, для интеграции WoT из форумом xenForo .
Что хочу сделать, это получить список состава клана и сравнить по имени, а лучше всего по id из дополнительным полем форума, в котором прописываем имя игрока.
Если имена совпадают, то добавляем пользователя в определенную группу форума, и это все должен делать этот скрипт, который будет запускаться через cron.
Если игрок вышел из клана, то скрипт должен удалить пользователя форума из определённой группы и перевести его в другую группу.
